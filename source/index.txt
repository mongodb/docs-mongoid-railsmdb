Introduction
------------

Welcome to the documentation site for {+cli-long+}. {+cli-short+} is a
command-line utility for creating, updating, managing, and maintaining Rails
applications that use Mongoid and MongoDB. {+cli-short+} is an extension of the ``rails``
command from {+framework+}.

Get Started
-----------

Learn how to install {+cli-short+}, establish a connection to MongoDB and begin
working with data in the :ref:`railsmdb-get-started` tutorial.
